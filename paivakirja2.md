# Oppimispäiväkirja: Hajautettu git

__Mikä osion tehtävissä oli vaikeaa ja mikä helppoa? Mikä auttoi minua oppimaan? Miten selvitin esteet, jotka vaikuttivat tehtävän suorittamiseen?__

Kävi niin, että luin materiaalit yhtenä päivänä ja jätin harjoituksen toiselle päivälle. Toisaalta huono juttu, sillä olin unohtanut paljon. Toisaalta hyvä, että tuli kerrattua materiaalia.

Haarojen ja repositorioiden välillä seikkailu tuntuu vielä vähän mystiseltä. Hieman vaikea hahmottaa, mutta uskon, että ajan kanssa ne tulevat tutuiksi. Samoin GitHub tuntuu edelleen jokseenkin vieraalta, vaikka sitä on jo aiemmilla kursseilla jonkin verran käytetty. Ohjeiden ja Googlen avulla kuitenkin pärjää.

Git ei tunnu enää ihan yhtä pelottavalta kuin ennen tätä kurssia.
